WiFi_Readme
================
The Ameba WiFi related APIs and examples are works based on Arduino WiFI
shield libraries (https://www.arduino.cc/en/Reference/WiFi).

These include,

| WiFi.cpp
| WiFi.h
| WiFiServer.cpp
| WiFiServer.h
| WiFiUdp.cpp
| WiFiUdp.h
